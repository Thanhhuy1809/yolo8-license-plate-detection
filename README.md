# yolo8-license-plate-detection
A lightweight YOLOv8-based license plate detection script that preprocesses images, resizes them to training resolution, runs inference, and saves the detection results with bounding boxes. Designed for Vietnamese license plate datasets, fully customizable, and easy to integrate into larger ALPR pipelines.
Dự án này triển khai pipeline nhận diện biển số tự động bằng YOLOv8.
Mã nguồn cho phép nạp mô hình đã huấn luyện, tự động resize ảnh về đúng kích thước training (640×640), chạy dự đoán và lưu lại ảnh kết quả với bounding box.
Phù hợp cho hệ thống ALPR, bài toán nhận dạng biển số xe máy – ô tô tại Việt Nam.
